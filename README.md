# Assignment Project

## Features
- Workshop Monitoring Dashboard (Real-time UI)
- Task Manager (Full-stack CRUD)

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js, Express
- Database: PostgreSQL

## Run Locally

### Backend
cd backend
node server.js

### Frontend
Open index.html in browser

## API Endpoints
- GET /workshop
- GET /tasks
- POST /tasks
- PUT /tasks/:id
